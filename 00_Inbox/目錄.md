# 00_Inbox 目錄

## 資料夾用途

暫存尚未整理、歸類不明、來源不完整，或需要使用者確認的新資料。

## 瀏覽重點

- 優先處理 `needs-review` 的筆記。
- 整理後依 PARA 移到 `10_Projects/`、`20_Areas/`、`30_Resources/` 或 `40_Archives/`。
- 使用 YouTube 或 PDF 工具擷取的原始資料，先停留在這裡作為 Capture 階段。
